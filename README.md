# 🌐 Multi-Branch Network Infrastructure Design Using Cisco Packet Tracer

> A scalable and secure enterprise network infrastructure designed and simulated using **Cisco Packet Tracer**, implementing VLAN segmentation, subnetting, inter-VLAN routing, multilayer switching, and network security for a multi-department engineering faculty.

---

## 📖 Project Overview

This project presents the design and implementation of a **multi-branch enterprise network** for an engineering faculty using **Cisco Packet Tracer**. The objective was to create a reliable, scalable, and secure network capable of supporting multiple academic departments while ensuring efficient communication, network segmentation, and future expansion.

The network architecture incorporates **VLAN technology**, **IP subnetting**, **inter-VLAN routing**, **multilayer switching**, and a dedicated surveillance network to simulate a real-world campus infrastructure.

# 🏛 Network Architecture

The network consists of multiple academic and administrative departments connected through a hierarchical network architecture.

### Included Departments

- 💻 Computer Engineering Department
- ⚙️ Mechanical Engineering Department
- 🏗 Civil Engineering Department
- ⚡ Electrical & Electronic Engineering Department
- 🧠 Information Systems Department
- 🏢 Administration Department
- 📹 CCTV Surveillance Network

Each department is assigned an independent VLAN to improve security, reduce broadcast traffic, and simplify network management.

# 🌍 IP Addressing

The IP addressing scheme was carefully designed using subnetting techniques while reserving approximately **30% additional address space** for future expansion.

The design provides:

- Efficient address utilization
- Easy scalability
- Simplified network administration

# 🔀 Routing

The project implements **Inter-VLAN Routing** using multilayer switching to allow communication between different departments while maintaining VLAN isolation.
